# Project Brief

## Project Overview
- **Name**: My Doctor AI
- **Type**: Desktop Application (Electron)
- **Core Purpose**: Virtual doctor consultation chat app with local AI
- **Target Users**: Individuals seeking health symptom checking, families, community leaders

## Key Goals
- Chat interface for AI-powered doctor consultations
- Session management for different conversations
- Profile system with contextual onboarding
- Local AI integration for privacy
- RAG support for uploading private medical notes

## Scope
- Profile management with context-based questions
- Dashboard with health insights and recent chats
- Sessions list (table view) for conversation management
- Chat interface with conversation history
- Tools panel for enabling integrations (e.g., clinic scheduling)
- Document manager with drag & drop for RAG

## Success Criteria
- UI-only placeholder for now (no real AI backend)
- Blue gradient theme (Slack-style)
- Phase-by-phase development with Git commits
