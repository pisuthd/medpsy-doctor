# Product Context

## Why This Project Exists
- Provide accessible virtual doctor consultation through AI chat
- Enable health symptom checking with personalized context
- Support multiple family members or community users
- Keep medical data private with local AI processing

## Problems It Solves
- Accessibility to health guidance without immediate doctor visit
- Managing health conversations for multiple family members
- Organizing health-related consultations over time
- Privacy concerns with local AI processing

## How It Should Work
1. App starts with model loading (progress indicator)
2. User selects or creates a profile
3. Profile selection prompts contextual questions (age, role, etc.)
4. Main interface shows dashboard, sessions, and chat
5. Users can chat with AI for health assistance
6. Simplified UI with no menubar (privacy-first approach)

## User Experience Goals
- Clean, professional blue gradient interface (Slack-style)
- Smooth animations with Framer Motion
- Intuitive navigation between dashboard/sessions/chat
- Free & Open Source messaging on dashboard
- Simplified settings with context size configuration
